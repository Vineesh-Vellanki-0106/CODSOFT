# Task-2: Basic Calculator Application (Python)

## Description

This project is a command-line based calculator developed using Python.
It performs basic arithmetic operations based on user input.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Floor Division
* Modulus
* Power

## Technologies Used

* Python

## How to Run

1. Install Python
2. Download the project
3. Run:

   ```
   python calculator.py
   ```

## Learning Outcome

* Understanding conditional statements
* Handling user input
* Implementing arithmetic logic
* Building command-line applications
