# Task-1: To-Do List Application (Python)

## Description

This project is a command-line based To-Do List application developed using Python.
It helps users create, manage, and track their daily tasks efficiently.

## Features

* Add new tasks
* View all tasks
* Update task status (mark as completed)
* Delete tasks
* Stores tasks using JSON file handling

## Technologies Used

* Python
* JSON
* File Handling
