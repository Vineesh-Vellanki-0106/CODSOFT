# Task-3: Password Generator (Python)

## Description

This project is a Python-based password generator that creates strong and random passwords based on user input.

## Features

* User-defined password length
* Multiple complexity options:

  * Letters only
  * Letters + Numbers
  * Letters + Numbers + Symbols
* Random and secure password generation

## Technologies Used

* Python
* random module
* string module

## How to Run

1. Install Python
2. Download the project
3. Run:

   ```
   python password_generator.py
   ```

## Learning Outcome

* Randomization techniques in Python
* Working with string libraries
* Building security-focused utilities
* User input handling
